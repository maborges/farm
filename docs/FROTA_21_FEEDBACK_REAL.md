## 2. Padrões Identificados

### Pontos Fortes
- **Headline de Resultado (B):** É unânime a atração pelo número "20% de redução".
- **Baixo Risco:** A microcopy "Sem cartão de crédito" remove a principal barreira de entrada para o teste gratuito.
- **Prova de Valor:** O exemplo do trator (R$ 8.000) gera identificação imediata com a dor financeira.

### Pontos de Dúvida / Confusão
- **Hardware vs Software:** Alguns visitantes confundiram o produto com um dispositivo físico de telemetria.
- **Setup:** "Configuração em 7 dias" soa rápido demais para alguns, gerando dúvida se é manual ou automático.
- **Conectividade:** Dúvida recorrente se o sistema funciona offline na fazenda.
